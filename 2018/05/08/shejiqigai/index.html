设计乞丐

发表于 2018-05-08 | 分类于 流水账 |

记录一下上一周。

周二晚回到米兰，本来以为这次算好了到达时间一定可以按时休息，不巧碰上罢工，的士uber全打不上，如果不是找到了一个华人司机，大概真要在雨中拉着行李箱走三公里回家了。

周三上班，前一天没睡好已经很累，结果学校突然需要我改cv和作品集，编辑也说需要重新改图注，最后做完已经凌晨四点。

周四上班，因为只睡了四小时一整天属于灵魂出窍的状态，随便吃了个饭回到家八点就开始睡，连续睡了十二个小时依然困，起来时发誓再也不熬夜。

周五上班，心想终于周五了今晚约个朋友吃饭吧，接着下班前临时接到任务，聚餐改成了见客户。客户没带齐资料，大概浪费了一个多小时在等，回到家继续干活，完成时已经两点了，说好的不熬夜也要为金钱低头。

我工作完时T君已经睡醒了，我说今天的客户人挺好，因为东西要得急还主动给我多付了钱。他说你开价太低了，你的东西怎么会只值这点钱。我说这个价格其实已经高于平均价了，现在你知道在意大利和中国创意民工有多穷了吧。他说：“你应该不会想知道我们公司的报价列表。”

我心想，别说你们这些搞投资的，就我今晚在店里听到两个店员聊天，其中一个在另一个咖啡店里兼职的姑娘每周只用上四天班月薪都比设计师要高。

学金融和学计算机的，都说自己是金融民工、码农，学设计的呢？大概该叫乞丐了。

# 日常 # 流水帐 # 没钱不要学设计
一月
佛系买卖
